# Day12：意図的な空白日（配信なし）

---

## ■ 配信なし

**配信条件：** なし（配信しない）
**目的：** 「不在感」の演出

---

### 設計意図

Day11で「明日は何も送らない」と予告済み。

**なぜ空白が効くのか：**

1. **10日間毎日届いていた配信が突然止まる** → 「あれ？今日来ない」という意識
2. **個人LINEだからこそ不在感が強い** → 公式LINEの不在は気にならないが、個人LINEの不在は気になる
3. **「本気で変わりたいか？」を考える時間** → 自分の中で結論を出させる
4. **飢餓感の創出** → Day13の配信への期待値が最大化

---

### 担当者のアクション（Day12にやること）

**朝（8:00-8:30）：**
- Day11の個人LINEメッセージへの反応チェック
- 「会ってみたい」系の返信があった人 → Day13を待たずにアポ調整してOK
- 返信なしでも焦らない。空白日は「待つ」のが正解

**夜（21:00）：**
- 配信なし。何もしない。
- 担当者自身が「何か送りたくなる」衝動を我慢する ← これが重要
