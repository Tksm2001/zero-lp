## ■ Day13 配信タイムライン（まとめ）

```
21:02  通1｜逆説的クロージング＋アポ打診（個人LINE・手動）
即時   → 「話したい」返信者にアポ日程調整（手動）
即時   → アポ確定者にフォロー一言（手動）
48h後  → 未返信者に最終フォロー（手動）
```

**この通の設計意図：**
- 逆説的クロージング：「無理にとは言わない」「焦る必要もない」
- 被害者データで最も効果的だった手法の正当版
  - 詐欺：「今日中に決めないとなくなる」→ 即断を迫る
  - 我々：「自分のペースで決めてね」→ 時間を与える
  - **この1点が決定的な違い**
- 「話したい」だけ返信すればいい = 行動のハードルを最小化
- 30分の電話 = 3回アポの初回への導線
