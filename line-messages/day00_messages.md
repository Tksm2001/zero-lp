## ■ 補足：Day0の加速ルート判定ロジック

**通3への返信を以下の基準で判定：**

### 加速ルート移行条件（3つ中2つ該当で移行）

| 条件 | 判定方法 |
|------|---------|
| ① 10分以内に返信 | Lステップの返信時間で自動判定可 |
| ② 3行以上の自由返信 | 担当者が目視で判定 → 手動タグ |
| ③ 感情を語っている | 担当者が目視で判定 → 手動タグ |

**判定後のアクション：**
- 2つ該当 → `route_加速` タグ付与 → Day1以降の自動配信を停止 → 手動チャットに移行
- 1つ以下 → `route_標準` タグ付与 → Day1以降の自動配信を継続

### ランク判定（Day0終了時点）

| ランク | 条件 | タグ |
|--------|------|------|
| A | 通3に2往復以上のやり取り成立 | `rank_A` |
| B | 通3に1回返信あり | `rank_B` |
| C | 既読のみ（返信なし） | `rank_C` |
| D | 未読（未開封） | `rank_D` |
