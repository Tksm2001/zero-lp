# 長期育成シナリオ（月2回×12ヶ月）

---

## ■ 概要

**対象：** Day13までにアポに至らなかったC/Dランク
**タグ条件：** `rank_C` or `rank_D`、かつアポ未確定
**配信元：** 公式LINE（自動配信）
**頻度：** 月2回（1日と15日）
**期間：** 12ヶ月（全24通）
**目的：** 「欲」を定期的に刺激し、再起動のタイミングを待つ

---

## ■ 設計思想

被害者データの最重要発見：
**「断っても1年間LINEで欲を刺激し続けて契約に至った」**

これの正当版を設計する。
- 押し売りしない
- 毎回価値を提供する
- 「変わりたい」が再燃した時に、思い出してもらえる存在でいる

---

## ■ 配信カレンダー（全24通）

### Month 1（登録翌月）

**1日｜成功者ストーリー①**
```

今日は1人紹介させて。

32歳、元事務職のMさん。
毎日9時-18時、手取り20万。

「このまま定年まで？」って
思った日にLINEをくれた人。

今は在宅で月40万。
朝の満員電車はもう乗ってない。

Mさんが最初にやったこと、
知りたかったら「知りたい」って送ってね。
```

**15日｜マインド系コンテンツ①**
```
こんばんは、あいりです。

「明日やろうは馬鹿やろう」

…って言葉、みんな知ってるけど、

本当に怖いのは
「明日やろう」が
「来月やろう」になって、

まだ生きてる？
```

## ■ 長期育成の反応対応ルール

**返信が来た場合：**
→ 担当者が24時間以内に手動返信
→ 温度が高ければDay5の内容から再教育開始
→ アポ希望なら即日程調整

**ブロックされた場合：**
→ 何もしない。正常な離脱。

**既読がつかなくなった場合：**
→ 3ヶ月連続未読 → 配信停止（コスト削減）
