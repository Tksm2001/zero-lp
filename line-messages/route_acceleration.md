# 加速ルート：手動チャットガイドライン

---

## ■ 概要

**対象：** Day0の返信で加速ルート判定された人（100人中10-15人）
**タグ：** `route_加速`
**運用：** 自動配信を **停止** し、全て手動チャットで対応
**目標：** 最短3日でアポ到達（通常13日）

---

## ■ 加速ルートの原則

1. **テンプレは使わない。** 相手の言葉に100%カスタムで返す
2. **チェックポイントの順番は守る。** 速度は自由、順番は絶対
3. **1日の上限なし。** 相手が話したいだけ話させる
4. **音声メッセージを早期投入。** Day0中に使ってもOK

---

## ■ 5つのチェックポイント通過ガイド

### CP① 共感成立
**確認サイン：** 相手が自分の話をし始めた
**担当者のアクション：**
- 相手の言葉をそのまま引用して返す
- 「わかる」「私もそうだった」を自然に使う
- 質問を重ねすぎない。2つ聞いたら1つ自分の話をする

**例文：**
```
「満員電車がしんどい」って言ってたけど、
私もあれ毎朝死んでた。笑

特に月曜の朝とかヤバくなかった？
```

→ 相手の言葉を引用（1行）→ 自分の共感（1行）→ 具体的な問い（1行）

---

### CP② 痛みに触れた
**確認サイン：** 現状への不満を言語化した
**担当者のアクション：**
- 不満を「正当化」する。「そう思って当然だよ」
- 深掘りしすぎない。痛みは触れるだけでいい
- 「その状況で頑張ってるの、すごいよ」で労う

**例文：**
```
うわ、それはしんどいね…。

でもその状況でちゃんと毎日やってるの、
普通にすごいと思う。

私だったら持たなかったかも。
```

---

### CP③ 解放を感じた
**確認サイン：** 「私のせいじゃないんだ」的な発言
**担当者のアクション：**
- Day4の「あなたのせいじゃない」メッセージの内容を会話で伝える
- 「才能じゃなくて環境の問題」を自然に織り交ぜる
- 相手が自分を責めていたら、全力で否定する

**例文：**
```
ちょっと待って。
それ、あなたのせいじゃないよ。

やり方がわからなかっただけ。
地図なしで歩いてたんだから
迷って当たり前じゃない？

自分を責めちゃダメ。
```

---

### CP④ 投資観が変わった
**確認サイン：** 「お金かけてでも変わりたい」的な発言
**担当者のアクション：**
- 自分の投資体験を具体的に話す
- 金額は出してもいいが、相手から聞かれた時だけ
- 「高い」ではなく「価値がある」のフレーミング

**例文：**
```
私も最初「え、そんなかかるの？」って思った。

でも、今の生活をあと5年続けるコストと、
ここで環境変えるコスト、
冷静に比べたら答えは出たんだよね。

「5年分の時間を買った」って
今は思ってる。
```

---

### CP⑤ 信頼が個人に移った
**確認サイン：** 「あいりさんがいうなら」「あいりさんに出会えてよかった」
**担当者のアクション：**
- ここまで来たら師匠の話を出す
- 「あなたみたいな人にこそ会ってほしい人がいる」
- アポの打診へ自然に移行

**例文：**
```
ありがとう。そう言ってくれて嬉しい。

実はね、私が変われたのは
ある人に出会えたから。

あなたみたいに本気の人にこそ
会ってほしい人がいるんだよね。

30分だけ、電話で話してみない？
```

## ■ 加速ルートのタイムライン例

```
Day0 15:00  登録＋PDF配布（自動）
Day0 15:01  欲求4択（自動）
Day0 15:03  思い出す問い（自動）
Day0 15:10  ★返信到着 → 加速判定
Day0 15:15  手動チャット開始（自動配信停止）
Day0 15:30  CP① 共感成立
Day0 16:00  音声メッセージ投入
Day0 21:00  やり取り継続 → CP② 痛みに触れた

Day1 08:00  朝のチェック→個別返信
Day1 21:00  CP③ 解放を感じた

Day2 12:00  昼のやり取り → CP④ 投資観
Day2 21:00  CP⑤ 信頼移転 → 師匠の話

Day3 08:00  アポ打診
Day3 12:00  アポ確定
```

**最短3日。通常ルートの1/4の期間。**
**アポ到達率：50-60%（通常ルートの5-10倍）**

---

## ■ 加速ルートのNG行動

| NG | 理由 |
|----|------|
| テンプレ返信 | 加速ルートの人は「特別扱い」に敏感。テンプレはバレる |
| 返信を急かす | 「今忙しい？」等は禁止。相手のペースに合わせる |
| 長文を送りすぎ | 1回の返信は3-5行まで。相手より長い返信をしない |
| 全CPを1日で通過 | 焦りすぎ。最低2日はかける。Day0で⑤まで行こうとしない |
| 商品の話をする | 金額・内容は絶対に言わない。アポで話す |
| 師匠を先に出す | CP⑤「信頼が個人に移った」を確認してから |
